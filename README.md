# md2do for Obsidian

Manage tasks in markdown with due dates, priorities, assignees, tags, and Todoist sync.

## Features

- Scan your vault for markdown tasks with rich metadata
- Task list view with grouping and sorting
- Autocomplete for `@assignee`, `!priority`, `#tags`, and `#due/` dates
- Diagnostics and warnings for task issues
- Automatic completion dates when checking off tasks

## Syntax

```markdown
- [ ] Task description @assignee !high #tag #due/2025-03-15
- [x] Completed task {completed:2025-03-10}
```

## Installation

Search for "md2do" in Obsidian's Community Plugins browser.
